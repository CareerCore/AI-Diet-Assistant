Food Nutrition Dataset (150+ Everyday Foods)

Overview:
This project is based on a food nutrition dataset containing nutritional values of everyday foods.
It can be used for data analysis, machine learning, and health-related applications.

NutriSmart — AI-Powered Food Nutrition Intelligence Platform
 
Project Type: Real-Time Data Science & Machine Learning Project

Domain: Health Tech / Food Analytics

Dataset: Food Nutrition Dataset — 205 Everyday Foods (USDA FoodData Central)

Tech Stack: 
Python, Scikit-learn, Pandas, Matplotlib, Seaborn, Random Forest, KMeans, PCA, RandomForestRegressor, LinearRegression, Pipeline, RandomForestClassifier, LogisticRegression, confusion_matrix, classification_report, r2_score, cross_val_score, StandardScaler,MinmaxScaler

Dataset Description:
- Entries: ~150–200 food items
- Type: Tabular data

This dataset provides clean, structured, and easy-to-use nutritional information for more than 200 commonly consumed foods, including fruits, vegetables, grains, dairy, beverages, snacks, and cooked dishes.

The data has been sourced from the USDA FoodData Central API, which is one of the most trusted open food-nutrition sources globally.

Client Requirement:

Client: NutriSmart Health Solutions Pvt. Ltd.

Industry: Digital Health & Wellness

NutriSmart is a health-tech startup building a consumer-facing app to help everyday users fitness enthusiasts, diabetics, weight-watchers, and nutritionists make smarter food choices. The client wants a data-driven ML backend to automate food classification, calorie prediction, and dietary clustering.

Features:
- food_name: Name of food
- category: Food category
- calories: Energy (kcal)
- protein: Protein (g)
- carbs: Carbohydrates (g)
- fat: Fat (g)
- iron: Iron (mg)
- vitamin_c: Vitamin C (mg)

Objectives:
- Analyze food nutrition
- Build ML models
- Create diet recommendation systems
